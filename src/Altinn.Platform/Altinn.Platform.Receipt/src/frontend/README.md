## Running the tests

### Lint checks

1. Navigate to the folder `frontend`.
2. Execute `yarn --immutable`. This step is only nescessary if you have not already done it, or if you change branches.
3. Execute `yarn run lint`.

### Unit tests

1. Navigate to the folder `frontend`.
2. Execute `yarn --immutable`. This step is only nescessary if you have not already done it, or if you change branches.
3. Execute `yarn run test`.
