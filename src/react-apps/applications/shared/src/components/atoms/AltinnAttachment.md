### Description

AltinnAttachment kan display one or several attachments

```jsx
const attachments = [
  {
    name: 'attachment.doc',
    url: 'http://link.to/attachment.doc',
    iconClass: 'reg reg-attachment',
  },
];
<AltinnAttachment
  attachments={attachments}
  listDisableVerticalPadding={true}
/>
```
