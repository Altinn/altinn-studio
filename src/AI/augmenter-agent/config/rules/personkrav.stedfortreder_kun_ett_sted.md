# Stedfortreder er ikke styrer/stedfortreder på annet sted

## Vurdering
Samme prinsipp som for styrer. Saksbehandler må sjekke P360.

* Stedfortreder er navngitt: **maa_undersokes**. Be saksbehandler sjekke P360. Nevn navnet.
* Stedfortreder ikke navngitt: **maa_undersokes**.

Bruk `path_value` for stedfortreders navn.
