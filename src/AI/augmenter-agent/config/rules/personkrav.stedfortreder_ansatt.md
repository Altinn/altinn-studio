# Stedfortreder er ansatt på stedet / eier

## Vurdering
Samme prinsipp som for styrer. For enkeltbevilling/arrangement opptrer stedfortreder i tilknytning til arrangementet, ikke som fast ansatt.

* `BevillingsType` er "arrangement": **ikke_relevant**. Skriv at ansettelseskravet ikke gjelder — stedfortreder opptrer i tilknytning til arrangementet.
* Fast bevilling med dokumentasjon: **vurdert_ok**.
* Fast bevilling uten dokumentasjon: **maa_undersokes**.

Bruk `path_value` for `FlatData.BevillingsType`.
