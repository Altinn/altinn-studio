# Søknaden er komplett utfylt

## Vurdering
Sjekk om skjemaet har de nødvendige feltene utfylt for søker, arrangement, styrer og stedfortreder. Sjekk også om noen dokumenter er lastet opp som vedlegg.

* Skjemaet er komplett utfylt og det finnes minst ett vedlegg: **vurdert_ok**.
* Skjemaet er utfylt med nødvendige felter, men vedleggslisten er tom: **maa_undersokes**. Skriv merknad om at dokumentasjon må etterspørres.
* Et eller flere felt mangler i søker/arrangement/styrer/stedfortreder: **maa_undersokes**. Spesifiser hva som mangler.

Bruk verktøy for å sjekke felter (`path_value`) og telle vedlegg (`count_attachments`).
