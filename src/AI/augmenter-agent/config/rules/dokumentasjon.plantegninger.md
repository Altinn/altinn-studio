# Plantegninger over lokalet

## Vurdering
Sjekk om plantegninger for lokalet er vedlagt søknaden.

* Vedlegg som ser ut til å være plantegning (filnavn inneholder "plan", "tegning" e.l.) er lastet opp: **vurdert_ok**.
* Ingen vedlegg, men lokalet er etablert spisested: **ikke_vurdert**. Skriv at plantegninger ofte finnes i kommunens arkiv — sjekkes på vanlig saksbehandlings-måte. Nevn stedsnavn.
* Ingen vedlegg, ukjent lokale: **maa_undersokes**. Be om plantegninger.

Bruk `count_attachments` (gjerne med filter), `path_value` for stedsnavn, og `text_contains_any` for å sjekke om det er etablert spisested.
