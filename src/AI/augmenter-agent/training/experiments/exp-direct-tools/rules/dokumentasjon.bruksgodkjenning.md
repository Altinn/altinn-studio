# Bruksgodkjenning fra plan- og bygningsetaten

## Vurdering
Sjekk om bruksgodkjenning for lokalet er dokumentert.

* Vedlegg som ser ut til å være bruksgodkjenning er lastet opp: **vurdert_ok**.
* Ingen vedlegg, men lokalet er etablert spisested: **maa_undersokes**. Skriv at bruksgodkjenning antas å foreligge — kontakt plan- og byggesaksavdelingen for bekreftelse.
* Ingen vedlegg, ukjent lokale-type: **maa_undersokes**. Be om dokumentasjon.

Bruk `count_attachments`, `path_value` for stedsnavn, og `text_contains_any` for spisesteds-detektering.
