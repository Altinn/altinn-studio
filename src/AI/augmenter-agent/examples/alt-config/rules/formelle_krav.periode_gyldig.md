# Permisjonsperioden er gyldig

## Vurdering
Permisjonsperioden må ha en startdato og en sluttdato, og sluttdato må være lik eller etter startdato.

* Begge datoer oppgitt og start ≤ slutt: **vurdert_ok**. Skriv merknad som siterer perioden.
* Start eller slutt mangler: **maa_undersokes**. Forklar hva som mangler.
* Start etter slutt: **vurdert_avslag**. Skriv merknad som siterer begge datoer.

Bruk `path_value` for `Permisjonsperiode.StartDato` og `Permisjonsperiode.SluttDato`.
