<Paragraph>
  StudioTableRemotePagination brings together Digdir Designsystemet's `Table` and `Pagination` components.
  It dynamically renders a table by passing an array of objects to the `column` and `row` props.

  Use this component when the pagination logic is handled remotely (for example an external API).

  The pagination can be disabled by removing the `pagination` prop.
</Paragraph>
